# ByteBrew Café Bill Generator

A beginner-level Python mini project that simulates a café billing system.

## Features

* Displays a café menu
* Validates menu items
* Automatically retrieves item prices using a Python dictionary
* Calculates subtotal based on quantity
* Applies an optional 10% discount
* Calculates 5% tax
* Generates a formatted bill summary
* Displays a thank-you message at the end

## Technologies Used

* Python 3

## Concepts Practiced

* Variables
* User Input
* Data Types
* Conditional Statements (if-else)
* Dictionaries
* Arithmetic Operations
* String Formatting

## Sample Menu

* Latte – Rs 150
* Cappuccino – Rs 100
* Mocha – Rs 120
* Macchiato – Rs 200

## Purpose

This project was developed as a fresher college mini project to practice Python fundamentals and understand how basic billing systems work.
